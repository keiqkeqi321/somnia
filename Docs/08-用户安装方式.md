# 08 - 用户安装方式

## 概述

Somnia 提供 4 种安装方式，覆盖不同用户环境：

| 方式 | 用户需要什么 | 一行命令 |
|------|-------------|---------|
| pip | Python 3.11+ | `pip install somnia` |
| 一键脚本 | curl / PowerShell | `curl \| bash` 或 `irm \| iex` |
| npx | Node.js 16+ + Python | `npx somnia` |
| Docker | Docker | `docker run -it somnia` |

---

## 方式一：pip install（推荐）

### 前提
- Python 3.11+ 已安装
- pip 可用（Python 自带）

### 安装

```bash
pip install somnia
```

### 使用

```bash
somnia              # 交互式 REPL
somnia chat "你好"   # 单次对话
somnia doctor       # 检查环境
somnia --help       # 查看帮助
```

### 升级

```bash
pip install --upgrade somnia
```

### 卸载

```bash
pip uninstall somnia
```

---

## 方式二：一键安装脚本

自动检测 Python 环境，没有则引导安装。

### macOS / Linux

```bash
curl -fsSL https://raw.githubusercontent.com/keiqkeqi321/learn-claude-code/OpenAgent_by_codex/OpenAgent/scripts/install.sh | bash
```

### Windows PowerShell

```powershell
irm https://raw.githubusercontent.com/keiqkeqi321/learn-claude-code/OpenAgent_by_codex/OpenAgent/scripts/install.ps1 | iex
```

### 脚本行为

```
运行脚本
   │
   ├─ 检测 Python 3.11+
   │    ├─ 找到 → 继续
   │    └─ 没找到 → 提示安装方式（brew/winget/apt），然后退出
   │
   ├─ pip install somnia
   │
   └─ 验证 somnia --help
```

---

## 方式三：npx

### 前提
- Node.js 16+ 已安装
- Python 3.11+ 已安装

### 使用

```bash
# 直接运行（首次会自动 pip install somnia）
npx somnia

# 全局安装
npm install -g somnia
somnia
```

### 工作原理

```
npx somnia
   │
   ├─ 下载 npm wrapper 包
   │
   ├─ postinstall.js 自动执行 pip install somnia
   │
   └─ bin/somnia.js → 检测 Python → python -m open_somnia [args]
```

---

## 方式四：Docker

### 前提
- Docker 已安装

### 使用

```bash
# 构建镜像
cd OpenAgent
docker build -t somnia .

# 交互式
docker run -it somnia

# 单次对话
docker run -it somnia chat "你好"

# 带 API Key
docker run -it -e ANTHROPIC_API_KEY=sk-xxx somnia
```

---

## 从源码安装（开发者）

```bash
git clone https://github.com/keiqkeqi321/learn-claude-code.git
cd learn-claude-code/OpenAgent
pip install -e .
```

> `-e` 表示可编辑模式（editable），修改代码后立即生效，无需重新安装。

---

## 安装验证

安装成功后运行：

```bash
somnia --version     # 查看版本号
somnia --help        # 查看帮助
somnia doctor        # 环境检查
```

## Python 安装指南

如果用户没有 Python：

| 系统 | 命令 |
|------|------|
| Windows | `winget install Python.Python.3.12` |
| macOS | `brew install python@3.12` |
| Ubuntu | `sudo apt update && sudo apt install python3.12` |
| Fedora | `sudo dnf install python3.12` |
| 通用 | https://www.python.org/downloads/ |

> ⚠️ Windows 安装时务必勾选 **"Add Python to PATH"**。
