## 项目结构

```
OpenAgent/
├── VERSION                    # 版本号单点维护（唯一版本源）
├── pyproject.toml             # Python 包配置（动态读取 VERSION）
├── LICENSE                    # MIT 许可证
├── Dockerfile                 # Docker 部署
├── .dockerignore
├── README.md                  # 项目说明 + 安装文档
├── CHANGELOG.md               # 版本变更记录
│
├── openagent/                 # Python 源码（内部模块名）
│   ├── __init__.py            # __version__（sync 脚本写入）
│   ├── __main__.py            # python -m openagent 入口
│   ├── cli/                   # CLI 相关
│   │   ├── main.py            # argparse 入口，注册 somnia 命令
│   │   ├── repl.py            # 交互式 REPL
│   │   ├── commands.py        # 子命令定义
│   │   └── prompting.py       # 提示词处理
│   ├── config/                # 配置
│   ├── runtime/               # 运行时
│   ├── tools/                 # 工具集
│   └── storage/               # 持久化
│
├── npm/                       # npm wrapper 包
│   ├── package.json           # npm 包配置（版本号 sync 写入）
│   ├── bin/openagent.js       # CLI 入口（检测 Python → 调用 somnia）
│   ├── postinstall.js         # 安装时自动 pip install somnia
│   ├── index.js
│   └── LICENSE
│
├── scripts/                   # 运维脚本
│   ├── release.ps1            # Windows 一键发版
│   ├── release.sh             # Linux/macOS 一键发版
│   ├── sync-version.ps1       # Windows 版本号同步
│   ├── sync-version.sh        # Linux/macOS 版本号同步
│   ├── install.ps1            # Windows 用户一键安装
│   ├── install.sh             # Linux/macOS 用户一键安装
│   ├── publish-pypi.sh        # PyPI 手动发布
│   ├── publish-npm.sh         # npm 手动发布
│   └── get2fa.py              # 2FA 验证码生成（不入库）
│
├── tests/                     # 测试
└── Docs/                      # 运维文档（本目录）
```

## 版本号位置（三处同步）

| 文件 | 说明 |
|------|------|
| `VERSION` | **唯一版本源**，手动修改这个文件 |
| `openagent/__init__.py` | `__version__`，由 sync-version 脚本写入 |
| `npm/package.json` | `version`，由 sync-version 脚本写入 |
| `pyproject.toml` | 通过 `[tool.setuptools.dynamic]` 动态读取 VERSION，无需手动改 |

## 发布渠道

| 渠道 | 地址 | 方式 |
|------|------|------|
| PyPI | https://pypi.org/project/somnia/ | `pip install somnia` |
| npm | https://www.npmjs.com/package/somnia | `npm install somnia` / `npx somnia` |
| Docker | 本地构建 | `docker build -t somnia .` |
| GitHub Release | https://github.com/keiqkeqi321/learn-claude-code/releases | CI 自动创建 |
